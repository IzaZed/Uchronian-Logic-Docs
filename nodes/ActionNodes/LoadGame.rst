Load Game (Node)
===========================================
Category: *Game*

Load a previously saved game (Feature in development)

Node Properties:
----------------

Load from
    File path, defaults to File Path\Data\

Inputs:
-------

Condition
    The input Condition

Slot
    The slot from which to load the game (must be previously saved)

Outputs:
--------

Done
    True if the node performed successfully, else False
