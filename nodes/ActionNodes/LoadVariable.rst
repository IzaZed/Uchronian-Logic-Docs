Load Variable (Node)
===========================================
Category: *Variables*

Lad a previously saved variable into the game

Node Properties:
----------------

Load from
    File path, defaults to File Path\Data\

Inputs:
-------

Condition
    The input Condition

Name
    Name of the variable. Defaults to "var"

Outputs:
--------

Done
    True if the node performed successfully, else False

Value
    The saved Value
