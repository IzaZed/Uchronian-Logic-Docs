Install Sub Tree (Node)¶

Category: Logic Tree

Adds a tree to an object

Inputs:¶

Condition
The input condition
Target Object
The object which to add the tree to
Tree Name
The name of the tree to add
Enabled
Wether the tree is set to be active when added

Outputs:¶

Done
True if the node performed successfully, else False
