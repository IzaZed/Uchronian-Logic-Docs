Move To (Node)
===========================================
Category: *Transformation*

Move object to a location on the shortest path.

Inputs:
-------

Condition
    The input Condition

Object
    The object to be moved

Target Location
    Target as vector [x, y, z]

Move as Dynamic
    Wether to move the object using physics

Speed
    Movement speed

Stop at Distance
    stop moving the object when this close to Target

Outputs:
--------

When Done
    True if the object reached destination, else False
