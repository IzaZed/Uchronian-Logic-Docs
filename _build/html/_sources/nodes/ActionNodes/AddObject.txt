Add Object (Node)
===========================================
Category: *Objects*

Adds the given object to the curent scene.

Inputs:
-------

Condition
    The input condition

Scene
    Name of the scene in which to look for the object

Name
    Name of the object to be added

Life
    Time after which the object gets removed again (not removed if 0)


Outputs:
--------

Added Object
    The added object
