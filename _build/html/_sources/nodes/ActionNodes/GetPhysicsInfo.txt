Get Physics Info (Node)
===========================================

Gets the information from KX_CharacterWrapper of given object (if the object is set to Character Physics)

Inputs:
-------

Condition
    The input condition (Optional)

Object  
    The object whose info should be grabbed

Outputs:
--------

Max Jumps
    The maximum amount of jumps the player can perform

Active Jump Count
    The current jump number, can be used to assign separate animations to jumps

Gravity
    The gravity applied to the object

On Ground
    If the player is on ground or not (Boolean)
