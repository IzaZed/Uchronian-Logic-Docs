Edit Armature Constraint (Node)
===========================================

Edits the constaint of a bone in an Armature object.

Inputs:
-------

Condition
    The input condition

Armature
    The Armature object

Constraint Name
    Name of the constraint

Enforced Factor
    Fraction of constraint effect that is enforced

Primary Target
    Name of the primary target for the constraint to be set

Secondary Target
    Name of the secondary target for the constraint to be set

Active
    Sets the constraint to be active or inactive

IK Weight
    Weight of the Inverse Kinematics constraint

IK Distance
    Distance the constraint is trying to maintain with target

Outputs:
--------

Done
    True if the node performed successfully, else False
