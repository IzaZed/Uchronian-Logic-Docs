Save Game (Node)
===========================================
Category: *Game*

Save the game's current state. *This feature is WIP, it should not be used in production!*

Node Properties:
----------------

Save To
    Save file path, defaults to *File Path\Saves*

Inputs:
-------

Condition
    The input Condition

Slot
    The save slot. there can be an infinite number if the game logic allows for it

Outputs:
--------

Done
    True if the node performed successfully, else False
