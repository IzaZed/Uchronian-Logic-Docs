Set Light Color (Node)
===========================================
Category: *Lights*

Set the emissive color of a light.

Inputs:
-------

Condition
    The input Condition

Light Object
    The light object of which to modify the RGB values

Red
    The intensity of the red comonent

Green
    The intensity of the green comonent

Blue
    The intensity of the blue comonent

Outputs:
--------

Done
    True if the node performed successfully, else False
