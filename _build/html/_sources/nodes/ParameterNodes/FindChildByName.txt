Get Child By Name (Node)
===========================================

This Node takes in the parent object and a string.
It then looks in the object's children if there is one with
the given name.

If so, the child object is returned.

Inputs:
-------

Parent
    this is the parent object whose children will be searched.

Name
    this is the name of the child object.

Outputs:
--------

Child
    if found, this will be the child object.

Parent
    the same object as selected as input *Parent*
